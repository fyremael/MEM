# Revision Template

Use this block at the top of any engineering doc.

```text
## Revision History
- YYYY-MM-DD: <summary of change>, reason: <why>, impact: <what changed operationally>, refs: <paths/commands>
```

Use this decision entry block in `DECISION_LOG.md`.

```text
## Decision (YYYY-MM-DD)
- Decision: GO / NO-GO / CONDITIONAL
- Scope:
- Rationale:
- Evidence:
- Risks:
- Follow-up required before next decision:
```
