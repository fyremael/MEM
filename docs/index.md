# MEM Documentation Hub

This documentation set supports research progression with a reproducible, inspectable implementation and clear operating signals.

## What This Covers

- Architecture and process flow for the memory-channel system.
- Executive decision artifacts for go/no-go readiness.
- Auto-generated API reference from package source.
- Auto-generated context snapshot from tracked benchmark and gate artifacts.
- Process and governance docs used for methodical delivery.

## Quick Start

```bash
python -m pip install -e ".[dev,docs]"
python scripts/build_docs.py
```

Then view the static site locally:

```bash
python -m mkdocs serve --config-file mkdocs.yml
```

## Auto-Updating Contract

API and context pages are generated by scripts:

- `python scripts/build_api_reference.py`
- `python scripts/generate_context_docs.py`

`python scripts/build_docs.py` runs both generators, then builds the MkDocs site.

Use `python scripts/build_docs.py --check` in CI or pre-PR verification to fail on stale generated docs.

Change history for docs system updates is tracked in `docs/changelog.md`.
